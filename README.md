# RASH-U-CGPA-CALCULATOR
RASH-U CGPA CALCULATOR is a user-friendly tool that helps students accurately compute CGPA. Developed by Usman Abdulrashid, a B.Sc. Cybersecurity (200 Level) student in the Department of Computer Science, Faculty of Physical &amp; Computing Sciences at Usmanu Danfodiyo University Sokoto.
